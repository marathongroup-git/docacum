## Repositorio
- **Código fuente** (C#, ASPX, .csproj): Git.
- **Customization ZIP**: guarda en Git LFS o artefact repository.
- **Playbooks**/RFC/Evidencias: Markdown + adjuntos.

## Evidencias mínimas
- Casos de prueba y resultados (QA/UAT).
- Capturas o logs (PXTrace), documentos exportados.
- Checks de performance y seguridad si aplica.
