## Plan Odoo: Una Aplicación Gratis (One App Free)

El proyecto se despliega bajo el modelo de licenciamiento **"Una aplicación gratis"** de Odoo. Este plan ofrece ventajas estratégicas significativas para la gestión focalizada de procesos:

*   **Acceso 100% Gratuito de por Vida**: Mientras se utilice únicamente un módulo (en este caso, **Inventario**) y sus dependencias técnicas automáticas, el uso de la plataforma no tiene costo de licencia.
*   **Usuarios Ilimitados**: A diferencia de los planes pagados que cobran por usuario, este plan permite registrar una cantidad **ilimitada** de usuarios sin costo adicional, facilitando la colaboración de todo el equipo.
*   **Infraestructura SaaS (Software as a Service)**: La instancia está alojada y mantenida directamente por Odoo en la nube (Odoo Online). Esto elimina la necesidad de mantener servidores propios, copias de seguridad manuales o actualizaciones de seguridad, garantizando una disponibilidad del servicio del 99.9%.
*   **Funcionalidades Enterprise**: Al estar en Odoo Online, se tiene acceso a la versión Enterprise del módulo seleccionado, con todas sus características avanzadas habilitadas.
